# GNN 기반 캠퍼스 건물 혼잡도 예측 시스템
> 성균관대학교 박호건 교수 수업 후반기 조별과제

---

## 프로젝트 개요

- **목표**: GNN(GAT)으로 캠퍼스 건물별 혼잡도 예측
- **데이터**: 성균관대 GLS 수강편람 2025년 1·2학기 수강인원
- **범위**: 소프트웨어학과 전공과목 개설 건물 7개 (자연과학캠퍼스)
- **평가 방식**: 1학기 학습 → 2학기 실제 데이터로 cross-semester 평가

---

## 그래프 설계

### 노드 (7개 건물)
| 동 | 이름 | 특징 |
|----|------|------|
| 21 | 정보통신대학 | 22동 남쪽 연결 |
| 22 | 제1공학관 | 클러스터 허브 |
| 23 | 공과대학 | 클러스터 북단 주진입구 |
| 26 | 제2공학관 | 25/26/27 클러스터 정문 |
| 33 | 화학관 | 주차장 남쪽 |
| 40 | 반도체관 | 33동 연결 복합동 |
| 85 | 산학협력센터 | 캠퍼스 북부 ㄷ자형 |

### 엣지 (12개 방향 엣지)
- 출입구 가중치 (주문 0.6 / 후문 0.2) + 이동 거리(m)
- 캠퍼스 주축: 85 ↔ 26 ↔ 23 ↔ 22 ↔ 21
- 남단: 22 ↔ 33 ↔ 40

## 모델 및 실험

### 태스크
- **입력**: 시각 t의 건물별 체류 인원 + 시간 피처 (sin/cos 인코딩, 5-dim)
- **출력**: 시각 t+1의 건물별 체류 인원 (one-step ahead prediction)

### 하이퍼파라미터
| 항목 | 값 |
|------|----|
| Hidden dim | 64 |
| Epochs | 200 |
| Optimizer | Adam, lr=1e-3 |
| Synthetic 인스턴스 | 200개 |
| Train/Val | 25,920 / 6,480 샘플 |
| max_occ (정규화 기준) | 569명 (1학기 기준 고정) |

### 실험 결과 (2026-05-26)

| 모델 | 1학기 val MAE | **2학기 test MAE** | val→test 격차 |
|------|:------------:|:-----------------:|:------------:|
| MLP  | 17.2명        | **32.2명 ✅**       | +15.0명      |
| GCN  | 15.4명        | 57.8명             | +42.4명      |
| GAT  | 21.6명        | 54.7명             | +33.1명      |

> **2학기 cross-semester 평가: MLP 최우수**  
> GCN/GAT는 1학기 그래프 흐름 패턴에 과적합 → 학기 바뀌면 오히려 악화

### 핵심 발견
- GNN이 소규모 그래프(7노드)에서 동일 학기 내에는 유효하나, 학기 간 스케줄 변화에 취약
- MLP는 시간대-건물 패턴 학습으로 cross-semester 일반화 더 강인

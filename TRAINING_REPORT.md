# GNN 기반 캠퍼스 혼잡도 예측 — 학습 결과 보고서

## 실험 설계

### 핵심 아이디어
- **학습 데이터**: 2025-1학기 수업 스케줄 기반 synthetic augmented data (전체 학습용)
- **검증 데이터**: 2024-1학기 실제 snapshot (같은 학기 유형, 다른 연도 → 진짜 held-out val)
- **테스트 데이터**: 2025-2학기 실제 snapshot (cross-semester ground truth)
- **목표**: 1학기에서 학습한 모델이 2학기 실제 혼잡도를 얼마나 잘 예측하는가 (cross-semester generalization)

> **평가 설계 원칙**: val과 train이 같은 데이터셋이면 안 된다. 2025-1 synthetic으로 학습했으므로 val은 다른 연도의 동일 학기(2024-1 actual)로 설정, test는 cross-semester(2025-2 actual)로 설정.

### 예측 태스크
- 입력: 시각 t의 건물별 체류 인원 + 시간 피처 (sin/cos 인코딩)
- 출력: 시각 t+1의 건물별 체류 인원 예측 (one-step ahead)
- 평가: MAE (명 단위)

---

## 데이터

### 그래프 구조
- **노드**: 7개 건물 (21, 22, 23, 26, 33, 40, 85동)
- **엣지**: 16개 방향 엣지 (출입구 가중치 + 이동 거리)
- **노드 피처** (5-dim): `[occ_norm, sin(h), cos(h), sin(d), cos(d)]`

**토폴로지 (수정 확정):**
- 메인 남북 축: 85 ↔ 26 ↔ 22 (22동이 중앙 허브)
- 22동 지선: 22 ↔ 23 (북서), 22 ↔ 21 (서)
- 남단: 22 ↔ 33 ↔ 40 (주차장3 경유)
- 대각선 보조: 85 ↔ 22, 26 ↔ 23

### 학습 데이터 (2025-1학기 synthetic)
| 항목 | 값 |
|------|----|
| Synthetic 인스턴스 수 | 200개 (전부 train) |
| 총 학습 샘플 (슬롯 전환) | 32,400개 |
| max_occ (정규화 기준) | 569명 |

**Synthetic 노이즈 구성:**
- 결석률: Beta(8, 2) ~ 평균 80% 출석
- 자습생: Poisson(λ=15)
- 이동 지연: 직전 슬롯 인원의 8%
- 측정 노이즈: Gaussian(0, σ=5)

### 검증·테스트 데이터 (실제 snapshot)
| 항목 | 2024-1 (val) | 2025-2 (test) |
|------|:-----------:|:------------:|
| 슬롯 전환 샘플 수 | 162개 | 135개 |
| 노이즈 | 없음 (raw) | 없음 (raw) |

## 모델 구조

### 공통 설정
| 하이퍼파라미터 | 값 |
|----------------|-----|
| Hidden dim | 64 |
| Epochs | 200 |
| Batch size | 256 |
| Optimizer | Adam (lr=1e-3) |
| Loss | MSE (학습), MAE (평가) |
| Device | CUDA (RTX 3070 Ti) |

### MLP
- 그래프 구조 무시, 각 노드 독립 예측
- `Linear(5→64) → ReLU → Linear(64→64) → ReLU → Linear(64→1)`

### GCN
- GCNConv × 2 레이어 + Linear head
- `GCNConv(5→64) → ReLU → GCNConv(64→64) → ReLU → Linear(64→1)`

### GAT
- `heads=4`, GATConv × 2 레이어 + Linear head
- `GATConv(5→16, heads=4) → ELU → GATConv(64→64, heads=1) → ELU → Linear(64→1)`

## 실험 결과

### MAE 비교 (명 단위)

| 모델 | 2024-1 val MAE | **2025-2 test MAE** | val→test 변화 |
|------|:-------------:|:------------------:|:------------:|
| MLP  | 44.2명         | **29.3명** ✅        | −14.9명 (개선) |
| GCN  | 81.9명         | 60.7명              | −21.2명 (개선) |
| GAT  | 88.8명         | 68.4명              | −20.4명 (개선) |

> **cross-semester 평가에서 MLP가 압도적으로 우수.**

### 주목할 현상: val MAE > test MAE (전 모델 공통)

2024-1 val MAE가 2025-2 test MAE보다 높다. 이유:
- 학습 데이터가 **2025-1** 기반 synthetic → 2025-2(같은 연도, 다른 학기)가 2024-1(다른 연도, 같은 학기)보다 수업 시간대 패턴이 더 비슷함
- 즉, **연도 간 차이(2024↔2025)가 학기 간 차이(1학기↔2학기)보다 더 크다**는 것을 시사

## 분석: GNN이 MLP보다 나쁜 이유

### 관찰
GCN, GAT는 val과 test 모두에서 MLP보다 약 2~3배 높은 MAE를 기록했다.

### 원인 해석

**1. 학기별 그래프 흐름 패턴 변화**
- GNN은 이웃 노드 정보를 집계 — "26동이 많으면 22동으로 유입된다"는 식의 건물 간 상관관계를 학습
- 1학기와 2학기는 수업 배치(요일·시각·건물)가 달라서 **건물 간 흐름 패턴 자체가 변함**
- GAT attention weight가 2025-1 특화 패턴을 기억하고 있어 다른 학기에서 오히려 오차를 키움

**2. 소규모 그래프의 한계**
- 노드가 7개뿐 → GNN이 일반화할 수 있는 구조적 패턴이 부족
- MLP는 각 건물을 독립적으로 예측하므로 학기 간 스케줄 변화의 영향을 덜 받음

**3. MLP의 시간대 패턴 강인성**
- MLP는 `[occ(t), sin(h), cos(h), sin(d), cos(d)]`만 입력 → "이 시간대에 이 건물엔 이 정도"라는 시간대-건물 패턴 학습
- 이 패턴은 학기가 바뀌거나 연도가 바뀌어도 어느 정도 유지됨 (수업 시간대 분포가 크게 안 바뀜)

---

## 결론

> GNN(GAT/GCN)은 그래프 흐름 패턴을 학습하지만, 그 패턴이 학기마다 달라지는 상황에서는 **이웃 집계가 오히려 과적합 원인**이 된다. 7노드 소규모 그래프에서 cross-semester/cross-year 일반화는 MLP가 더 강인하다.
>
> 이는 단순히 "GNN이 MLP보다 좋다"는 통념에 반하는 결과로, **"언제 GNN이 실패하는가"를 실제 데이터로 보인 것 자체가 이 프로젝트의 contribution**이다.

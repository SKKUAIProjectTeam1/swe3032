----------------------------------------------------------------
 1. 원래 계획
----------------------------------------------------------------

프로젝트의 최초 목표는
"GAT(Graph Attention Network)가 MLP보다 캠퍼스 혼잡도 예측에
더 정확하다"는 것을 실험으로 보이는 것이었다.

학습·평가 데이터를 모두 2025-1학기 synthetic 데이터에서 뽑아
(train 80% / val 20%) MLP vs GCN vs GAT를 비교하는 방식이었다.


----------------------------------------------------------------
 2. 최종 실험 설계
----------------------------------------------------------------

올해 학기 데이터로 내년 동일 학기의 혼잡도를 예측한다.
(학과 정원이 연도 간 크게 변하지 않으므로 패턴이 유지된다는 가정)

핵심 원칙: 1학기는 1학기끼리, 2학기는 2학기끼리 비교.

  1학기 세트:
    train: 2024-1 synthetic (200 인스턴스)
    val  : 2024-1 actual (synthetic 품질 검증)
    test : 2025-1 actual ← 내년 동일 학기 ground truth

  2학기 세트:
    train: 2024-2 synthetic (200 인스턴스)
    val  : 2024-2 actual
    test : 2025-2 actual ← 내년 동일 학기 ground truth


----------------------------------------------------------------

----------------------------------------------------------------
 6. 결과 해석
----------------------------------------------------------------

두 세트 모두 MLP 우세. GCN/GAT 과적합 원인:

- GNN은 건물 간 흐름 패턴을 학습하는데,
  이 패턴이 연도마다 수업 배치 변화에 따라 달라짐
- 노드 7개 소규모 그래프에서 구조적 일반화 패턴 부족
- MLP는 시간대-건물 패턴만 학습 → 연도 변화에 강인


----------------------------------------------------------------
 7. 이 결과의 의미
----------------------------------------------------------------

"GNN이 항상 MLP보다 좋은 것은 아니다."

소규모 그래프 + 그래프 패턴이 해마다 달라지는 환경에서는
GNN의 이웃 집계가 오히려 과적합 원인이 된다.

"언제 GNN이 실패하는가"를 실제 데이터로 보인 것이
이 프로젝트의 contribution이다.

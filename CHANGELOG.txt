----------------------------------------------------------------
 1. 원래 계획
----------------------------------------------------------------

프로젝트의 최초 목표는
"GAT(Graph Attention Network)가 MLP보다 캠퍼스 혼잡도 예측에
더 정확하다"는 것을 실험으로 보이는 것이었다.

학습·평가 데이터를 모두 2025-1학기 synthetic 데이터에서 뽑아
(train 80% / val 20%) MLP vs GCN vs GAT를 비교하는 방식이었다.


----------------------------------------------------------------
 2. 방향 전환: 크로스 셈스터 평가 설계
----------------------------------------------------------------

같은 학기 데이터를 train/val로만 나누면 평가가 약하다.
synthetic noise로 만든 데이터끼리 비교하면 결국 "노이즈
패턴을 얼마나 잘 외웠냐"를 측정하는 것에 가까워지기 때문이다.

더불어 train/val이 같은 데이터셋이면 과적합 탐지가 안 된다.
올바른 설계는 아래와 같다:

  학습(train): 2025-1학기 synthetic augmented data (200 인스턴스)
  검증(val)  : 2024-1학기 실제 snapshot (같은 1학기, 다른 연도)
  테스트(test): 2025-2학기 실제 snapshot (cross-semester ground truth)

이렇게 하면 두 가지를 동시에 측정할 수 있다:
  - val MAE: 연도 간 일반화 (2025-1 학습 → 2024-1 예측)
  - test MAE: 학기 간 일반화 (2025-1 학습 → 2025-2 예측)

 5. 최종 실험 결과 (2026-05-28)
----------------------------------------------------------------

  모델   2024-1 val MAE   2025-2 test MAE
  -----  --------------  ---------------
  MLP       44.2명          29.3명  ← 최우수
  GCN       81.9명          60.7명
  GAT       88.8명          68.4명

주목할 현상 — val MAE > test MAE (전 모델):
  2024-1(다른 연도, 같은 학기)이 2025-2(같은 연도, 다른 학기)보다
  2025-1 학습 데이터와 패턴 차이가 더 크다.
  → 연도 간 차이가 학기 간 차이보다 더 큼을 시사.

이전 실험(train/val 모두 synthetic 2025-1)과 비교:
  MLP  32.2명 → 29.3명 (개선)
  GCN  57.8명 → 60.7명 (유사)
  GAT  54.7명 → 68.4명 (소폭 악화)


----------------------------------------------------------------
 6. 결과 해석
----------------------------------------------------------------

GCN/GAT가 cross-semester에서 나빠진 이유:
- GNN은 이웃 집계 구조상 건물 간 흐름 패턴을 학습한다.
- 이 패턴은 학기마다 수업 배치가 달라지면서 함께 변한다.
- GAT attention weight가 2025-1 특화 패턴을 기억하고 있어
  다른 학기·연도에서 오히려 예측을 방해했다.

MLP가 강인한 이유:
- MLP는 각 건물을 독립적으로 예측.
  "이 시간대에 이 건물엔 이 정도"라는 시간대-건물 패턴을 학습.
- 이 패턴은 학기·연도가 바뀌어도 어느 정도 유지된다.


----------------------------------------------------------------
 7. 이 결과의 의미
----------------------------------------------------------------

"GNN이 항상 MLP보다 좋은 것은 아니다."

노드가 7개뿐인 소규모 그래프에서, 그래프 패턴 자체가
학습/테스트 간에 달라지는 상황이라면 GNN의 이웃 집계가
오히려 과적합 원인이 된다.

이는 GNN 도입의 전제 조건("그래프 구조가 안정적으로 유지된다")이
cross-semester/cross-year 설정에서 만족되지 않기 때문이다.

이 결과 자체가 프로젝트의 contribution이다:
단순히 "GNN이 좋다"를 보이는 것보다, "언제 GNN이 실패하는가"를
실제 데이터로 보인 것이 더 설득력 있는 실험이다.

# GNN 기반 캠퍼스 혼잡도 예측 — 학습 결과 보고서

## 실험 설계

### 핵심 아이디어
- **학습 데이터**: 2025년 1학기 수업 스케줄 기반 synthetic augmented data
- **테스트 데이터**: 2025년 2학기 실제 snapshot (ground truth)
- **목표**: 1학기에서 학습한 모델이 2학기 실제 혼잡도를 얼마나 잘 예측하는가 (cross-semester generalization)

### 예측 태스크
- 입력: 시각 t의 건물별 체류 인원 + 시간 피처 (sin/cos 인코딩)
- 출력: 시각 t+1의 건물별 체류 인원 예측 (one-step ahead)
- 평가: MAE (명 단위)

---

## 데이터

### 그래프 구조
- **노드**: 7개 건물 (21, 22, 23, 26, 33, 40, 85동)
- **엣지**: 12개 방향 엣지 (출입구 가중치 + 이동 거리)
- **노드 피처** (5-dim): `[occ_norm, sin(h), cos(h), sin(d), cos(d)]`

### 학습 데이터 (2025-1학기)
| 항목 | 값 |
|------|----|
| 원본 snapshot 슬롯 수 | — |
| Synthetic 인스턴스 수 | 200개 |
| 총 학습 샘플 (슬롯 전환) | 25,920개 |
| Train / Val 분할 | 80% / 20% → 25,920 / 6,480 |
| max_occ (정규화 기준) | 569명 |

**Synthetic 노이즈 구성:**
- 결석률: Beta(8, 2) ~ 평균 80% 출석
- 자습생: Poisson(λ=15)
- 이동 지연: 직전 슬롯 인원의 8%
- 측정 노이즈: Gaussian(0, σ=5)

### 테스트 데이터 (2025-2학기)
| 항목 | 값 |
|------|----|
| 실제 snapshot 슬롯 수 | 140 슬롯 (요일×시각) |
| Test 샘플 (슬롯 전환) | **135개** (노이즈 없음, raw ground truth) |

## 모델 구조

### 공통 설정
| 하이퍼파라미터 | 값 |
|----------------|-----|
| Hidden dim | 64 |
| Epochs | 200 |
| Optimizer | Adam (lr=1e-3) |
| Loss | MSE (학습), MAE (평가) |

### MLP
- 그래프 구조 무시, 각 노드 독립 예측
- `Linear(5→64) → ReLU → Linear(64→64) → ReLU → Linear(64→1)`

### GCN
- GCNConv × 2 레이어 + Linear head
- `GCNConv(5→64) → ReLU → GCNConv(64→64) → ReLU → Linear(64→1)`

### GAT
- `heads=4`, GATConv × 2 레이어 + Linear head
- `GATConv(5→16, heads=4) → ELU → GATConv(64→64, heads=1) → ELU → Linear(64→1)`

## 실험 결과

### MAE 비교 (명 단위)

| 모델 | 1학기 val MAE | **2학기 test MAE** | 격차 (과적합) |
|------|:------------:|:-----------------:|:------------:|
| MLP  | 17.2명        | **32.2명** ✅       | +15.0명      |
| GCN  | 15.4명        | 57.8명             | +42.4명      |
| GAT  | 21.6명        | 54.7명             | +33.1명      |

> **2학기 cross-semester 평가에서 MLP가 가장 우수.**

## 분석: GNN이 MLP보다 나쁜 이유

### 관찰
GCN, GAT는 1학기 val에서 MLP보다 좋거나 비슷했지만, 2학기 실제 데이터에서는 MAE가 약 2배로 증가했다.

### 원인 해석

**1. 학기별 그래프 흐름 패턴 변화**
- GNN은 이웃 노드 정보를 집계 — "26동이 많으면 23동으로 유입된다"는 식의 건물 간 상관관계를 학습
- 1학기와 2학기는 수업 배치(요일·시각·건물)가 달라서 **건물 간 흐름 패턴 자체가 변함**
- GAT attention weight가 1학기 특화 패턴을 기억하고 있어 오히려 오차를 키움

**2. 소규모 그래프의 한계**
- 노드가 7개뿐 → GNN이 일반화할 수 있는 구조적 패턴이 부족
- MLP는 각 건물을 독립적으로 예측하므로 학기 간 스케줄 변화의 영향을 덜 받음

**3. 정보 누수 없는 MLP의 강인성**
- MLP는 `[occ(t), sin(h), cos(h), sin(d), cos(d)]`만 입력 → "이 시간대에 이 건물엔 이 정도"라는 시간대-건물 패턴 학습
- 이 패턴은 학기가 바뀌어도 어느 정도 유지됨 (수업 시간대 분포가 크게 안 바뀜)

---

## 결론 및 제언

### 결론
> GNN(GAT/GCN)이 동일 학기 내에서는 MLP와 비슷하거나 나은 성능을 보이지만, **학기 간 cross-semester 평가에서는 그래프 구조 학습이 과적합을 유발**해 MLP보다 나빠진다.

### 개선 방향
1. **Temporal GNN** — 시계열 구조(LSTM/Transformer)를 GNN과 결합해 시간 패턴 명시적 학습
2. **Cross-semester 학습 데이터 확보** — 여러 학기 데이터를 함께 학습 (현재는 1학기 synthetic만 사용)
3. **엣지 dropout 정규화** — GNN 과적합 완화
4. **그래프 피처 일반화** — 절대 인원 대신 상대 비율(건물 수용 인원 대비) 사용

# GNN 기반 캠퍼스 혼잡도 예측 — 학습 결과 보고서

## 실험 설계

### 핵심 아이디어
GNN을 이용해 **올해 학기 데이터로 내년 동일 학기의 혼잡도를 예측**한다.  
학과 정원은 연도 간 크게 변하지 않으므로, 시간대·건물별 패턴이 유지된다는 가정 하에 cross-year generalization을 평가한다.

> **평가 원칙**: 1학기는 1학기끼리, 2학기는 2학기끼리 비교.  
> train과 val이 같은 데이터셋이면 과적합 탐지가 불가능하므로,  
> val은 train의 실제 snapshot(synthetic 품질 검증용), test는 다음 연도 동일 학기로 설정.

### 예측 태스크
- 입력: 시각 t의 건물별 체류 인원 + 시간 피처 (sin/cos 인코딩)
- 출력: 시각 t+1의 건물별 체류 인원 예측 (one-step ahead)
- 평가: MAE (명 단위)

---

## 데이터

### 그래프 구조
- **노드**: 7개 건물 (21, 22, 23, 26, 33, 40, 85동)
- **엣지**: 16개 방향 엣지 (출입구 가중치 + 이동 거리)
- **노드 피처** (5-dim): `[occ_norm, sin(h), cos(h), sin(d), cos(d)]`

**토폴로지:**
- 메인 남북 축: 85 ↔ 26 ↔ 22 (22동이 중앙 허브)
- 22동 지선: 22 ↔ 23 (북서), 22 ↔ 21 (서)
- 남단: 22 ↔ 33 ↔ 40 (주차장3 경유)
- 대각선 보조: 85 ↔ 22, 26 ↔ 23

### 실험 세트 구성

| 구분 | Train (synthetic) | Val (actual) | Test (actual) |
|------|:----------------:|:-----------:|:------------:|
| **1학기 세트** | 2024-1 × 200 인스턴스 | 2024-1 실제 | **2025-1 실제** |
| **2학기 세트** | 2024-2 × 200 인스턴스 | 2024-2 실제 | **2025-2 실제** |

| 항목 | 1학기 세트 | 2학기 세트 |
|------|:--------:|:--------:|
| train 샘플 수 | 32,400개 | 32,400개 |
| val 샘플 수 | 162개 | 162개 |
| test 샘플 수 | 162개 | 135개 |
| max_occ (정규화 기준) | 980명 | 473명 |

**Synthetic 노이즈 구성:**
- 결석률: Beta(8, 2) ~ 평균 80% 출석
- 자습생: Poisson(λ=15)
- 이동 지연: 직전 슬롯 인원의 8%
- 측정 노이즈: Gaussian(0, σ=5)

## 모델 구조

### 공통 설정
| 하이퍼파라미터 | 값 |
|----------------|-----|
| Hidden dim | 64 |
| Epochs | 200 |
| Batch size | 256 |
| Optimizer | Adam (lr=1e-3) |
| Loss | MSE (학습), MAE (평가) |
| Device | CUDA (RTX 3070 Ti) |

### MLP
- 그래프 구조 무시, 각 노드 독립 예측
- `Linear(5→64) → ReLU → Linear(64→64) → ReLU → Linear(64→1)`

### GCN
- GCNConv × 2 레이어 + Linear head
- `GCNConv(5→64) → ReLU → GCNConv(64→64) → ReLU → Linear(64→1)`

### GAT
- `heads=4`, GATConv × 2 레이어 + Linear head
- `GATConv(5→16, heads=4) → ELU → GATConv(64→64, heads=1) → ELU → Linear(64→1)`

## 실험 결과

### 1학기 세트 — Train: 2024-1 synthetic / Test: 2025-1 actual

| 모델 | 2024-1 val MAE | **2025-1 test MAE** |
|------|:-------------:|:------------------:|
| MLP  | 29.5명         | **35.3명** ✅        |
| GCN  | 36.5명         | 72.9명              |
| GAT  | 61.6명         | 63.4명              |

### 2학기 세트 — Train: 2024-2 synthetic / Test: 2025-2 actual

| 모델 | 2024-2 val MAE | **2025-2 test MAE** |
|------|:-------------:|:------------------:|
| MLP  | 22.2명         | **33.8명** ✅        |
| GCN  | 25.7명         | 51.5명              |
| GAT  | 29.2명         | 46.3명              |

> **두 세트 모두 MLP가 cross-year 예측에서 가장 우수.**  
> 2학기 세트에서는 GAT(46.3명)가 GCN(51.5명)보다 나은 결과를 보임.

## 분석: GNN이 MLP보다 나쁜 이유

### 관찰
두 실험 세트 모두에서 GCN, GAT의 test MAE가 MLP 대비 약 1.5~2배 높다.  
특히 1학기 세트에서 GCN의 과적합이 두드러짐(val 36.5명 → test 72.9명).

### 원인 해석

**1. 연도 간 건물 흐름 패턴 변화**
- GNN은 이웃 노드 정보를 집계 → "26동이 많으면 22동으로 유입된다"는 건물 간 상관관계 학습
- 연도마다 수업 배치(요일·시각·건물)가 바뀌어 **건물 간 흐름 패턴 자체가 변함**
- GAT attention weight가 2024년 특화 패턴을 기억해 2025년 예측 시 오차 유발

**2. 소규모 그래프의 한계**
- 노드 7개 → GNN이 일반화할 수 있는 구조적 패턴 부족
- MLP는 각 건물 독립 예측으로 연도 간 스케줄 변화의 영향 최소화

**3. MLP의 시간대 패턴 강인성**
- 입력: `[occ(t), sin(h), cos(h), sin(d), cos(d)]` → "이 시간대에 이 건물엔 이 정도" 패턴 학습
- 이 패턴은 연도가 바뀌어도 어느 정도 유지됨 (수업 시간대 분포가 크게 안 바뀜)

---

## 결론

> 7노드 소규모 캠퍼스 그래프에서 **cross-year 예측(올해 → 내년 동일 학기)** 시,  
> GNN(GAT/GCN)의 이웃 집계는 연도 특화 패턴에 과적합되어 MLP보다 나쁜 성능을 보인다.  
> MLP는 그래프 구조를 무시하지만, 시간대-건물 패턴의 강인성 덕분에 cross-year 일반화에 유리하다.
>
> **"언제 GNN이 실패하는가"를 실제 데이터로 보인 것이 이 프로젝트의 contribution이다.**

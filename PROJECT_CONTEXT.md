# GNN 기반 캠퍼스 건물 혼잡도 예측 시스템
> 성균관대학교 박호건 교수 수업 후반기 조별과제

---

## 프로젝트 개요

- **목표**: GNN(GAT)으로 캠퍼스 건물별 혼잡도 예측
- **데이터**: 성균관대 GLS 수강편람 3년치 수강인원 (크롤링 완료)
- **범위**: 소프트웨어학과 전공과목 개설 건물부터 시작 (자연과학캠퍼스)

---

## 그래프 설계

### 노드
| 타입 | 단위 | 주요 속성 |
|------|------|-----------|
| `building` | 건물(동) 단위 | 건물명, 동 번호, 캠퍼스 좌표 |
| `entrance` | 건물 출입문 | 가중치(주출입문/후문/비상구), 방향 |

> 강의실 단위는 사용 안 함 (호실 정보 없음) → 건물(동) 단위로 뭉뚱그림

### 엣지
- 건물 간 인접도 / 이동경로
- 문별 가중치 (주출입문 높음, 후문 낮음) → **인위적으로 설정**
- 시간대(수업 시작/종료 시간) 반영 예정

### 모델
- **GAT (Graph Attention Network)**
- 입력: 시간대별 건물 수강인원 합산
- 출력: 건물별 혼잡도 예측값

## 데이터 현황

### 크롤링 완료: `gls_courses_raw.csv`
| 컬럼 | 설명 |
|------|------|
| 학수번호 | 과목 코드 (예: CSE3036-41) |
| 교과목명 | 과목명 (한글) |
| 담당교수 | 교수명 |
| 캠퍼스 | 자연과학 / 인문사회 |
| 개설학년도 | 2023 / 2024 / 2025 |
| 개설학기 | 1학기 / 2학기 |
| 개설강좌수 | 강좌 수 |
| 수강인원합 | 실제 수강인원 수 ← **핵심 피처** |
| 1학년(%) ~ 재수강(%) | 학년별 비율 |

### 누락된 데이터 (재크롤링 필요)
- **강의실/호실 정보** → 건물 단위로 대체하므로 불필요
- **수업 시간대** (예: 월15:00-15:50) → **재크롤링 필요**
  - GLS 목록 `cell_0_10` 컬럼에 존재 확인됨
  - 현재 CSV에 미포함

## 크롤러 현황

### 파일: `gls_crawler_playwright.py`
- **프레임워크**: Playwright (Selenium → 교체, Chrome 148 버전 호환 문제)
- **사이트**: `kingoinfo.skku.edu/gaia/nxui/index.html` (넥사크로/XFDL)
- **로그인**: 수동 로그인 후 Enter → 자동화 진행
- **파싱 방식**: 넥사크로 div ID 패턴 직접 파싱
  ```
  목록: [id*="gridrow_{i}.cell_{i}_{col}:text"]
  팝업: [id*="popupFrame"][id*="gridrow_{i}.cell_{i}_{col}:text"]
  ```
- **가상스크롤 대응**: `.Div.div_bodyArea_bg` 컨테이너 scrollTop 조작

### 확인된 컬럼 인덱스 (목록 그리드)
| cell 인덱스 | 내용 |
|------------|------|
| cell_X_3 | 학수번호 |
| cell_X_4 | 교과목명 |
| cell_X_5 | 담당교수 |
| cell_X_6 | "보기" 버튼 (팝업 트리거) |
| cell_X_7 | 캠퍼스 |
| cell_X_10 | **수업 시간대** ← 현재 미수집 |

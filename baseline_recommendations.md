# 캠퍼스 도로망 자동 설계 프로젝트: 추가 베이스라인 제안

GNN (GAT), MLP, GraphSAGE 외에 본 도로망 예측(에지 분류) 도메인에 적용 가능한 또 다른 베이스라인 모델 및 접근 방식들을 정리한 문서입니다.

---

## 1. 머신러닝 트리기반 기법 (LightGBM / XGBoost)
격자 형태의 그래프(Subgraph)에서 추출한 에지는 각 단일 행(Row)으로 이루어진 테이블 형식으로 변환하기 쉽습니다.

*   **피처 구성**:
    *   에지의 소스 노드 피처(9차원) + 대상 노드 피처(9차원)
    *   두 피처의 절대적 차이(Absolute difference) 및 내적(Element-wise product)
    *   에지 길이(좌표 간 유클리드 거리) 및 에지 고유의 `edge_ridge` 점수
*   **모델**: LightGBM Classifier / XGBoost Classifier
*   **장점**:
    *   매우 적은 학습 리소스 및 속도 (몇 초 이내 완료)
    *   각 피처의 중요도(Feature Importance) 분석을 통해 모델이 예측을 위해 어떤 피처(예: ridge, gate까지의 거리 등)에 의존하는지 파악 가능
    *   GNN이나 복잡한 딥러닝 없이 정형 데이터 분류 기법이 보여줄 수 있는 강력한 Baseline 기준점 수립

---

## 2. GCN (Graph Convolutional Network)
기본적인 GCN은 학습이 필요한 어텐션 메커니즘을 배제하고 연결된 노드들 간의 가중평균만으로 그래프 정보를 전파합니다.

*   **동작 방식**: 
    $$H^{(l+1)} = \sigma(\tilde{D}^{-\frac{1}{2}} \tilde{A} \tilde{D}^{-\frac{1}{2}} H^{(l)} W^{(l)})$$
*   **구현 방법**: 
    `MHGATLayer` 대신 노드의 인접 차수(Degree)를 계산하고, 가중평균 메시지 집계를 단순 Matrix multiplication 연산이나 PyTorch의 `scatter_add_`를 통해 단순화하여 구현합니다.
*   **장점**:
    *   파라미터 수 및 계산량의 대폭 감소
    *   Attention 과적합(Overfitting) 현상을 억제하며 격자 구조 그래프 상에서 부드러운 임베딩 전파 가능

---

## 3. 비학습 기반 물리적/휴리스틱 알고리즘 (Dijkstra / A*)
머신러닝을 아예 사용하지 않고, 맵에서 수집한 정보를 기반으로 경로를 구성하는 규칙 기반(Rule-based) 베이스라인입니다.

*   **동작 방식**:
    *   각 에지의 비용(Cost)을 `1.0 / (edge_ridge + epsilon)` 또는 `dist_to_building`을 반영하여 정의합니다.
    *   선택된 게이트 노드와 건물 클러스터 중심(Terminal 노드들)을 시작점과 끝점으로 설정하여 Dijkstra 또는 A* 알고리즘을 사용해 서로를 잇는 최단 비용 경로를 모두 찾아 도로망을 구성합니다.
*   **장점**:
    *   학습용 데이터셋(OSM GT)이 전혀 없는 상황에서도 즉각 적용 및 시각화 가능
    *   "기계학습 모델들이 최소한 단순 탐색 알고리즘보다 높은 연결성과 영리함을 보이고 있는가?"를 검증하는 가장 기본적인 물리적 대조군(lower bound) 역할을 수행

---

## 4. 좌표 기반 뉴럴 필드 (Coordinate-based Neural Field)
도로망은 캠퍼스의 지리적인 패턴(2D 공간)에 크게 종속됩니다. 에지 및 노드의 도메인 피처를 모두 무시하고 순수 좌표값만으로 도로의 연결 확률을 매핑하는 공간 메모리 회귀 모델입니다.

*   **동작 방식**:
    *   입력: 에지 양 끝점의 2D 정규화 좌표 `[x_src, y_src, x_dst, y_dst]` (4차원)
    *   모델: 중간에 ReLU를 포함한 4-5층의 순수 MLP
    *   출력: 에지 확률 (1차원 Sigmoid)
*   **장점**:
    *   모델이 주변 환경 피처(ridge, distance 등)를 전혀 보지 못하고 특정 위치 좌표에만 의존해 경로를 암기(Memorization)하는 성능이 어느 정도인지 파악하여 데이터셋의 공간적 편향(Spatial Bias)을 파악하는 데 유용
